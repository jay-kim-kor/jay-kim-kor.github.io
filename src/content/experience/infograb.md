## 금융권 DevOps 플랫폼 구축

인터넷전문은행의 DevOps 전환을 위한 통합 플랫폼 구축 프로젝트를 수행하고 있습니다.

### 주요 업무

- **Observability 스택 구축**: Grafana, Loki, Prometheus 기반 통합 모니터링 시스템 설계 및 구축
- **CI/CD 파이프라인 고도화**: GitLab Runner 기반 표준 빌드 파이프라인 아키텍처 설계, 10+ 프로젝트 적용
- **형상관리 마이그레이션**: SVN → Git 전환으로 병렬 개발 환경 도입, 엔터프라이즈 브랜치 전략 수립
- **시스템 통합**: 기존 ITSM 시스템과의 API 연동 인터페이스 설계
- **워크플로우 자동화**: n8n, Keycloak 기반 자동화 도구 통합

### 기술적 특이점

- 금융권 규제 준수를 위한 엔터프라이즈급 CI/CD 파이프라인 설계
- 레거시 형상관리(SVN) 기반 조직의 Git 전환 전략 수립 및 실행
- ITSM 연동을 위한 API 인터페이스 명세 및 호출 구조 설계
