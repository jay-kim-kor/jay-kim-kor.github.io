## 프로젝트 개요

클라우드 MSP에서 20개 이상 기업의 온프레미스 인프라를 Microsoft 365와 Azure로 전환한 프로젝트입니다. Active Directory 마이그레이션, 메일 서버 전환, 보안 위협 대응까지 포함합니다.

## 주요 작업

### 클라우드 마이그레이션
20개 이상 기업의 온프레미스 인프라(AD, 메일 서버, 파일 서버)를 Microsoft 365 및 Azure로 전환했습니다. 기업별 환경 분석, 마이그레이션 계획 수립, 실행, 검증까지 End-to-End로 수행했습니다.

### 하이브리드 클라우드 통합
온프레미스 Active Directory를 Azure AD와 연동하는 하이브리드 구성을 설계하고 구축했습니다. Azure AD Connect를 활용한 디렉토리 동기화와 SSO 환경을 구성했습니다.

### 보안 위협 대응
사내 AD 서버를 대상으로 한 CLDAP reflection DDoS 공격을 감지하고 차단했습니다. 네트워크 트래픽 분석을 통해 공격 패턴을 식별하고 방어 규칙을 적용했습니다.
